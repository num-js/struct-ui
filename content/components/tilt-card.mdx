## Installation

```bash
npx shadcn@latest add "https://ui-struct.vercel.app/r/tilt-card"
```

## Usage

```tsx
import TiltCard from "@/components/ui/tilt-card";

<TiltCard className="w-80 h-96 rounded-2xl">
  <div className="w-full h-full bg-gradient-to-br from-blue-600 to-purple-700">
    Your content here
  </div>
</TiltCard>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Card content |
| `className` | `string` | `''` | Container CSS classes |
| `tiltMaxAngle` | `number` | `15` | Maximum tilt angle in degrees |
| `perspective` | `number` | `1000` | CSS perspective value |
| `scale` | `number` | `1.02` | Scale factor on hover |
| `speed` | `number` | `400` | Animation spring stiffness |
| `glare` | `boolean` | `false` | Enable glare overlay effect |
| `glareMaxOpacity` | `number` | `0.35` | Maximum glare opacity |
| `glareColor` | `string` | `'white'` | Glare color |
| `borderGlow` | `boolean` | `false` | Enable border glow effect |
| `borderGlowColor` | `string` | `'rgba(255,255,255,0.5)'` | Border glow color |
| `disabled` | `boolean` | `false` | Disable tilt effect |

## Mobile Responsiveness

The tilt effect works best on desktop with mouse interaction. For mobile devices:
- The card displays normally without tilt
- Consider using `disabled={true}` on small screens
- All content remains fully accessible

```tsx
// Example: Disable on mobile
const isMobile = typeof window !== 'undefined' && window.innerWidth < 768;

<TiltCard disabled={isMobile}>
  ...
</TiltCard>
```

## Combining Effects

You can combine multiple effects for richer interactions:

```tsx
<TiltCard
  glare={true}
  glareMaxOpacity={0.3}
  borderGlow={true}
  borderGlowColor="rgba(168, 85, 247, 0.5)"
  tiltMaxAngle={12}
  scale={1.05}
>
  ...
</TiltCard>
```
