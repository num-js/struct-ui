## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Button content |
| `className` | `string` | `''` | Additional CSS classes |
| `particleCount` | `number` | `30` | Number of particles |
| `particleSize` | `[number, number]` | `[4, 10]` | Min/max particle size |
| `particleColors` | `string[]` | Rainbow colors | Array of particle colors |
| `particleShape` | `'circle' \| 'square' \| 'star' \| 'heart' \| 'mixed'` | `'mixed'` | Particle shape |
| `burstDirection` | `'radial' \| 'upward' \| 'explosion'` | `'radial'` | How particles spread |
| `duration` | `number` | `800` | Animation duration (ms) |
| `spread` | `number` | `120` | How far particles travel |
| `variant` | `'default' \| 'gradient' \| 'outline' \| 'neon' \| 'glass'` | `'default'` | Button style |
| `onClick` | `() => void` | - | Click handler |
| `disabled` | `boolean` | `false` | Disable button |

## Custom Colors

Create themed particle effects:

```tsx
// Ocean theme
<ParticleButton
  particleColors={['#0EA5E9', '#06B6D4', '#22D3EE', '#67E8F9']}
  particleShape="circle"
>
  Ocean Wave
</ParticleButton>

// Fire theme
<ParticleButton
  particleColors={['#EF4444', '#F97316', '#FBBF24', '#FCD34D']}
  particleShape="star"
>
  Fire Burst
</ParticleButton>
```

## Performance Tips

1. Keep `particleCount` under 50 for smooth animations
2. Lower `duration` for snappier feedback
3. Use `spread` wisely - larger values need more particles
